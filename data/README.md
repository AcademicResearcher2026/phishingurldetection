Raw datasets are not redistributed in this repository.

1. PhiUSIIL Phishing URL (Website) Dataset
   Download from UCI Machine Learning Repository:
   https://archive.ics.uci.edu/dataset/967/phiusiil%2Bphishing%2Burl%2Bdataset
   Save the file as:
   data/raw/phiusiil.csv

2. Malicious URLs Dataset
   Download from Kaggle:
   https://www.kaggle.com/datasets/sid321axn/malicious-urls-dataset
   Save the file as:
   data/raw/malicious_urls.csv

After downloading the datasets, run the scripts in numerical order.
